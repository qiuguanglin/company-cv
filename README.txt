#how to run
- open Main.java and run directly, supposedly console pops up with instructions to follow

#how to run the tests
- open BookingServiceTest.java to execute the test
- or use mvn test

#notice
- this is a maven project, it contains Junit dependencies in the pom.xml
- you can import Junit 4.12 jar into the project's classpath, if you are not using any framework support
- in the code lambda expressions are used, meaning java8+ is required to recognize it
